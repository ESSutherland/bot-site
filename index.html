General Commands

Command Name Command Info Command Format
Bot Get information about the bot. bot
Help Get information about commands. help
WhoIs Get information about the user. whois [@User] - ([]=optional)
Lookup Lookup a discord user via user ID. lookup {User ID} - ({}=required)
Enable Enables a feature module in the bot. enable {Module Name} - ({}=required)
Disable Disables a feature module in the bot. disable {Module Name} - ({}=required)
Reload Reloads a feature module in the bot. reload {Module Name} - ({}=required)
Modules List all modules and their status. modules [Page] - ([]=optional)
SetRole Assign the specified role to a defined role variable in the bot. setrole {Role Name} {@Role} - ({}=required)
SetChannel Assign the specified role to a defined role variable in the bot. sechannel {Channel Name} {#Channel} - ({}=required)
Status Change the playing status of the bot to the specified message. status {Message} - ({}=required)

Color Commands

Command Name Command Info Command Format
Color Allows Nitro Boosters to change the color of their name in the server. color/colour {Hex} - ({}=required)

Level Commands

Command Name Command Info Command Format
Level Displays the current level of the user or mentioned user(s). level [@User] - ([]=optional)
LevelTop Displays the top 5 users with the highest level. leveltop
SetExp Set the amount of experience that each level takes. setexp {Exp Amount} - ({}=required)

Minecraft Commands

Command Name Command Info Command Format
Whitelist Allows subscribers to whitelist themselves on the linked Minecraft server. whitelist {MC Username} - ({}=required)
SetRCON Allows subscribers to whitelist themselves on the linked Minecraft server. setrcon {IP} {Port} {Password} - ({}=required)

Custom Command Commands

Command Name Command Info Command Format
Commands Shows list of the custom commands on the server. commands
Command Create a custom command to use in the server. Flags: -a = everyone, -b = boosters, -s = subs, -m = mods, @ a user for a private command. command {Command Name} {Permission Flag} {Response} - ({}=required)
Delete Delete a custom command from the server. delete {Command Name} - ({}=required)
