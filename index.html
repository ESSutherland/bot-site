Commands
Highlighted commands are only usable by members with certain roles.
  • SUBSCRIBERS
  • BOOSTERS
  • MODERATORS
  • ADMINS

General Commands

Command Name Command Info Command Format
Bot Get information about the bot. bot
Help Get information about commands. help
WhoIs Get information about the user. whois {@User}
Lookup Lookup a discord user via user ID. lookup {User ID}
Ban Ban a user from the server using their discord ID. ban {User ID} {Reason}
MassBan Ban multiple users from the server using a username. massban {Username} {Reason}
AutoBan Add a username to a list that will be auto banned upon joining the server. autoban {Username}
UnAutoBan Remove a username form the auto ban list. unautoban {Username}
Enable Enables a feature module in the bot. enable {Module Name}
Disable Disables a feature module in the bot. disable {Module Name}
Reload Reloads a feature module in the bot. reload {Module Name}
Modules List all modules and their status. modules
SetRole Assign the specified role to a defined role variable in the bot. setrole {Role Name} {@Role}
SetChannel Assign the specified channel to a defined channel variable in the bot. sechannel {Channel Name} {#Channel}
Status Change the playing status of the bot to the specified message. status {Message}

Color Commands

Command Name Command Info Command Format
Color Allows Nitro Boosters to change the color of their name in the server. color {Hex}

Level Commands

Command Name Command Info Command Format
Level Displays the current level of the user. level
Leaderboard Displays leaderboard of player levels in the server. leaderboard
SetExp Set the amount of experience that each level takes. setexp {Exp Amount}

Minecraft Commands

Command Name Command Info Command Format
Whitelist Allows subscribers to whitelist themselves on the linked Minecraft server. whitelist {MC Username}
SetRCON Allows subscribers to whitelist themselves on the linked Minecraft server. setrcon {IP} {Port} {Password}

Custom Command Commands

Command Name Command Info Command Format
Commands Shows list of the custom commands on the server. commands
Command Create a custom command to use in the server. command {Command Name} {Permission Flag} {Response}
Delete Delete a custom command from the server. delete {Command Name}
