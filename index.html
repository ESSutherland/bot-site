Commands
Commands in BOLD and BLUE are only usable by server moderators.

General Commands

Command Name Command Info Command Format
Bot Get information about the bot.
Help Get information about commands.
WhoIs Get information about the user.
Purge Purge a specified number of message by the user or tagged user out of the last 100 messages.
Module Enable or disable feature modules in the bot.
Modules List all modules and their status.
Lookup Lookup a discord user via user ID.
Stop Stops the bot (Don't use this).

Color Commands

Command Name Command Info Command Format
Color Allows Nitro Boosters to change the color of their name in the server.

Level Commands

Command Name Command Info Command Format
Level Displays the current level of the user or mentioned user(s).
LevelTop Displays the top 5 users with the highest level.
AddExp Adds the specified amount of experience to the user or mentioned user.

Minecraft Commands

Command Name Command Info Command Format
Whitelist Allows subscribers to whitelist themselves on the linked Minecraft server.

Custom Commands Commands

Command Name Command Info Command Format
Commands Shows list of the custom commands on the server.
Command Create a custom command to use in the server. Flags: -a = everyone, -b = boosters, -s = subs, @ a user for a private command.
Update Update the response to a custom command.
Delete Delete a custom command from the server.

Animal Crossing Commands

Command Name Command Info Command Format
Villager Get information about the specified Animal Crossing: New Horizons villager.
