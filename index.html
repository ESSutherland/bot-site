Commands
Commands in BOLD and BLUE are only usable by server moderators.

General Commands

Command Name Command Info Command Format
Bot Get information about the bot. bot
Help Get information about commands. help
WhoIs Get information about the user. whois [@user] - ([]=optional)
Purge Purge a specified number of message by the user or tagged user out of the last 100 messages. purge {amount} [@user] - ({}=required, []=optional)
Module Enable or disable feature modules in the bot. module {name} {enable/disable} - ({}=required)
Modules List all modules and their status. modules
Lookup Lookup a discord user via user ID. lookup {user ID} - ({}=required)
Stop Stops the bot (Don't use this). stop

Color Commands

Command Name Command Info Command Format
Color Allows Nitro Boosters to change the color of their name in the server. color/colour {hex} - ({}=required)

Level Commands

Command Name Command Info Command Format
Level Displays the current level of the user or mentioned user(s). level [@user] - ([]=optional)
LevelTop Displays the top 5 users with the highest level. leveltop
AddExp Adds the specified amount of experience to the user or mentioned user. addexp {amount} [@user] - ({}=required, []=optional)

Minecraft Commands

Command Name Command Info Command Format
Whitelist Allows subscribers to whitelist themselves on the linked Minecraft server. whitelist {MC username} - ({}=required)

Custom Command Commands

Command Name Command Info Command Format
Commands Shows list of the custom commands on the server. commands
Command Create a custom command to use in the server. Flags: -a = everyone, -b = boosters, -s = subs, @ a user for a private command. command {permission flag} {command} {response} - ({}=required)
Update Update the response to a custom command. update {command} {response} - ({}=required)
Delete Delete a custom command from the server. delete {command} - ({}=required)

Animal Crossing Commands

Command Name Command Info Command Format
Villager Get information about the specified Animal Crossing: New Horizons villager. villager {name} - ({}=required)
